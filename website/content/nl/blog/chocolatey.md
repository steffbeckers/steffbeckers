## Installatie

https://chocolatey.org/install

## Nuttige commando's

Controleer op verouderde geïnstalleerde packages:

```powershell
choco outdated
```

Upgrade alle geïnstalleerde packages in één keer:

```powershell
choco upgrade all -y
```
